# Tugas-Besar-Struktur-Data-Kelompok-1-Data-Penjualan-
Tugas besar praktikum mata kuliah Struktur Data kelompok 1 dengan judul "Data Pejualan". Implementasi multi linked-list yang memodelkan data toko dan data barang beserta relasi keduanya. Suatu toko bisa menjual banyak barang. Dan suatu barang bisa dijual oleh banyak toko.
